# Project: WindVoyager


### Description:
WindVoyager is a real-time balloon flight path visualization system that tracks high-altitude balloons using live WindBorne Systems data. It overlays weather conditions such as wind speed, temperature, and cloud cover, allowing users to explore how atmospheric factors influence balloon trajectories. Built with Node.js (Express.js) and JavaScript (React.js), WindVoyager dynamically updates to provide the latest 24-hour flight history. Future expansions will include air traffic overlays, pollution tracking, and machine learning-based trajectory predictions.
